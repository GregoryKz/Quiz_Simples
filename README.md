Quiz de TI Interativo

Projeto de um quiz educacional focado em Tecnologia da Informação, permitindo que os usuários respondam perguntas de múltipla escolha e recebam feedback imediato sobre suas respostas.

Funcionalidades

Perguntas de múltipla escolha sobre TI, HTML, CSS, JavaScript, sistemas operacionais, bancos de dados e mais.

Feedback imediato após cada resposta, indicando se está correta ou incorreta.

Barra de progresso indicando o avanço do quiz.

Resultado final mostrando a quantidade de respostas corretas.

Design responsivo e moderno, com cores suaves e destaque em respostas corretas/incorretas.

Tecnologias Utilizadas

HTML5 – Estrutura do quiz.

CSS3 – Estilização do quiz e efeitos visuais.

JavaScript – Lógica do quiz, validação de respostas, barra de progresso e resultado final.

Estrutura do Projeto
/assets
  ├─ style.css       # Estilos do quiz
  └─ script.js       # Lógica e interatividade do quiz
index.html            # Página principal com o quiz
README.md             # Documentação do projeto

Como Usar

Abra o arquivo index.html em qualquer navegador moderno.

Leia cada pergunta e selecione a opção desejada.

Clique em Responder para enviar sua resposta.

Após cada resposta, o quiz exibirá se você acertou ou errou.

Ao final, será exibido o resultado total do quiz.
